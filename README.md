# ✈️ Moza Cockpit Flight Profiles

Welcome to this repository. It contains pre-configured profile presets for
the Moza cockpit, intended for Microsoft Flight Simulator setups.

## 📂 File Index

Each entry includes the filename (linked to the file), controller, MSFS
version, and a short description to help you choose the right profile.

| Filename | Controller | MSFS Version | Description |
|---|---|---|---|
| [ab6-a320-msfs2024.preset](profiles/ab6-a320-msfs2024.preset) | AB6 | 2024 | Airbus A320 profile preset for the Moza cockpit. |
| [ab6-cessna-172-msfs2024.preset](profiles/ab6-cessna-172-msfs2024.preset) | AB6 | 2024 | Cessna 172 profile preset for the Moza cockpit. |
| [ab6-piston-msfs2024.preset](profiles/ab6-piston-msfs2024.preset) | AB6 | 2024 | Generic piston aircraft profile preset for the Moza cockpit. |

## 🎮 How to Use These Files

Import the `.preset` files from the `profiles` folder into your Moza Cockpit.

Feel free to contribute improvements or additional Moza cockpit profiles.
